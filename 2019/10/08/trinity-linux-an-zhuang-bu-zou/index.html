Trinity Linux 安装步骤

Trinity Version：4.1.1.233
Java Version：JAVA1.8

一、安装java环境

# root用户
1、下载jdk
https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
2、解压jdk并安装
tar -xvzf jdk-8u221-linux-x64.tar.gz -C /usr/
cd /usr
mv jdk-1.8.0_221 java
3、配置环境变量
vim /etc/profile

export JAVA_HOME=/usr/share/java
export JRE_HOME=/usr/share/java/jre
export CLASSPATH=.:${JRE_HOME}/lib/rt.jar:${JAVA_HOME}/lib/dt.jar:${JAVA_HOME}/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
4、生效配置
source /etc/profile
5、检验Java是否生效
java -version

二、安装PG（Postgresql）

# root用户下
1、运行安装文件
./postgresql-9.5.10-1-linux-x64.run --mode text

2、选择安装目录
Please specify the directory where PostgreSQL will be installed.
Installation Directory [/opt/PostgreSQL/9.5]:(按 Enter 使用预设)

3、选择数据存储位置
Please select a directory under which to store your data.
Data Directory [/opt/PostgreSQL/9.5/data]:(按 Enter 使用预设)

4、输入数据库的密码
Please provide a password for the database superuser (postgres).
Password :trinity
Retype password : trinity

5、选择端口号
Please select the port number the server should listen on.
Port [5432]: (按 Enter 使用预设)

6、选择语言
[16] en_US.utf8
Please choose an option [1] :16

7、确认开始安装
Setup is now ready to begin installing PostgreSQL on your computer.
Do you want to continue? [Y/n]:Y

8、完成安装
Please wait while Setup installs PostgreSQL on your computer.
Installing
0% ______________ 50% ______________ 100% #########################################
-------------------------------------------------------------------- Setup has finished installing PostgreSQL on your computer.

9、取消 Launch Stack Builder
Launch Stack Builder at exit?
Stack Builder may be used to download and install additional tools, drivers and applications to complement your PostgreSQL installation. [Y/n]:n

三、初始化Trinity数据库

# root用户
su - postgres
/opt/PostgreSQL/9.5/bin/psql -h ip

1、创建user
CREATE USER trinity WITH PASSWORD 'trinity';
2、创建数据库
CREATE DATABASE trinity OWNER trinity;
3、切换用户测试
/opt/PostgreSQL/9.2/bin/psql -U trinity

四、执行Trinity安装程序

# trinity用户
1、执行安装程序
java -jar TrinityInstaller-v4.1.1.233.jar -console

2、选择安装的产品
选取需要安装的产品，输入对应的产品号码(默认为全选)，当产品被选取之后，会显示*号，输入 0 进行下一个步骤.。
Trinity Product List:
[*]1. JCS Server
[*]2. JCS Agent
[*]3. DIS Server
[*]4. Trinity Repository
press 0 to next step.
Please select install trinity product:0

3、输入安装位置
Please input install Product Path(Default: /home/Users/Trinity):

4、选取所要使用的 IP，这里的范例选择 127.0.0.1。
IP List:
[ ] 1. 192.168.10.129
[ ] 2. 127.0.0.1
Please select IP, press 0 to next step:2

5、连接PG数据库
Please input PostgreSQL info:

Port(5432):
Use default: 5432
Database Name(trinity):
Use default: trinity
User Name(trinity):
Use default: trinity
password(trinity):

Is drop table?[y/n]
Use defeclt: y
UI Server Port[8080]:

6、进行安装

Trinity DIS Service
